# calendar-activities-backend
--------------------
Created Using spring initializer using the following configurations:
- Project: Gradle
- Language: Java
- Spring Boot: Latest Stable Version (2.7.2)
- Java Version: 11
- Dependencies:
    * Spring Web
    * Spring Security
    * Spring Data JPA
    * Rest Repositories
    * Lombok
    * Validation
    * Postgres Driver
-DB Connection: `jdbc:postgresql://localhost:5432/calendar-app`
